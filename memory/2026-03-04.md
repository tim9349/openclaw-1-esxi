### 5. 修正 Config 驗證錯誤
OpenClaw 會對 Config 進行嚴格驗證，常見錯誤及修正：

#### 錯誤 1：`groups.allowlist` 格式錯誤
**錯誤格式**：
```json
"groups": {
  "allowlist": {
    "-1003783648951": true
  }
}
```

**正確格式** (必須是陣列)：
```json
"groups": {
  "allowlist": [
    "-1003735178483",
    "-1003783648951"
  ]
}
```

#### 錯誤 2：缺少頂層 `allowFrom`
**錯誤訊息**：
```
channels.telegram.groupPolicy is "allowlist" but groupAllowFrom (and allowFrom) is empty
```

**修正方法**：在 `channels.telegram` 頂層加入 `allowFrom`：
```json
"telegram": {
  "enabled": true,
  "dmPolicy": "pairing",
  "allowFrom": [
    "1120737982"
  ],
  "groupPolicy": "allowlist",
  // ...
}
```
